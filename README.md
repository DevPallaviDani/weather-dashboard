# 🌤 Weather Dashboard

A modern and responsive Weather Dashboard built using React and Tailwind CSS. This project displays current weather conditions, a 7-day forecast, and temperature trends in a clean and user-friendly interface.

## ✨ Features

- 🌡 Current weather display
- 📅 7-day forecast
- 🔍 City search UI
- 🎨 Modern dashboard UI design
- 📱 Responsive layout

## 🚀 Tech Stack

- React.js
- Tailwind CSS
- Vite
